# Test Case Design using XMind

This repository contains test design artifacts created using XMind.

## Login Module Test Design
The XMind file includes:
- Positive test scenarios
- Negative test scenarios
- Boundary condition tests
- UI validation cases
- Priority markers (High/Medium/Low)
- Execution status markers (Pass/Fail)

Tool Used:
- XMind (Free version)

Purpose:
- Visual test design
- Requirement coverage
- Review with QA team
